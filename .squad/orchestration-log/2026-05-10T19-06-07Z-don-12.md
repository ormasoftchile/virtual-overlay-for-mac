## Key Decisions Ratified

1. **Live preview without Apply button:** Changes publish immediately to `WatermarkAppearance`, which `OverlayController` subscribes to. Native macOS preference patterns expect no Apply button; state is saved to disk on debounce + app termination.

2. **Separate preferences.json:** Independent from spaces.json (identity store). Appearance schema evolution and Space-name persistence have different risk profiles and timelines.

3. **Design language swatches:** 6 curated colors match Susan's app icon palette (warm off-white, near-black), ensuring visual cohesion across the product.

4. **500ms debounce:** Slider drags remain responsive; persistent I/O doesn't thrash the filesystem during interactive adjustment.
